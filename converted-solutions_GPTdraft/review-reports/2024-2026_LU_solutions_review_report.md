# LEET 언어이해 공식 해설 Markdown 검수 보고서 (2024-2026)

## 검수 개요

- 검수 대상 연도: 2024, 2025, 2026학년도
- 과목: 언어이해(LU)
- 원문 비교 대상:
  - `법전협_2024 LEET 해설_언어이해.pdf`
  - `법전협_2025 LEET 해설_언어이해.pdf`
  - `법전협_2026 LEET 해설_언어이해.pdf`
- 검수 대상:
  - `2024_LU_official+solutions.md`
  - `2025_LU_official+solutions.md`
  - `2026_LU_official+solutions_package.zip` 내 `2026_LU_official+solutions.md` 및 이미지 2개
- 검수 방식: 원문 PDF 전 페이지를 이미지로 렌더링하여 문항별 해설 구간, 문항번호, 정답 표시, 선택지 해설, 표/그림 포함 여부를 대조하였다. PDF가 이미지 스캔본이므로 OCR은 보조적으로 사용했고, 판독이 애매한 문자 오류는 PDF 렌더링 이미지에서 직접 확인하였다.
- 참고: 문제 원문, 제시문, <보기>, 선택지 자체는 변환 대상에서 제외하라는 전제가 있으므로, 원문 PDF의 문제 영역에만 있는 표ㆍ그림ㆍ그래프는 누락으로 보지 않았다. 다만 해설 영역 안에 들어간 표ㆍ그림ㆍ캡션은 검수 대상으로 보았다.

# 2024학년도

## 검토요약

- 원문 PDF: 86쪽 스캔본.
- Markdown: 문항별 해설 `01`부터 `30`까지 모두 존재한다.
- 해설 본문 자체는 대체로 원문의 흐름을 반영하고 있으나, **전 문항의 명시적 정답 표시가 누락**되어 있다. 원문 PDF의 해설 블록에는 각 문항마다 `정답 : ①/②/...` 형식의 정답 정보가 들어 있으나, Markdown에는 `정답 :` 항목이 0개이다.
- 원문 PDF의 `정답 해설`, `오답 해설` 라벨은 Markdown에서 `선택지별 해설` 아래 선택지 번호별 설명으로 통합되어 있다. 내용 이해에는 큰 지장이 없지만, 원문 해설의 구조 정보를 보존한다는 관점에서는 정답 선택지와 오답 선택지를 명시적으로 구분하는 편이 바람직하다.
- 해설 내 별도 이미지 파일이 필요한 도표는 확인되지 않았다. 원문 PDF 중 제시문/문제 원문에 포함된 표ㆍ그림은 변환 제외 대상으로 판단하였다.

## 오탈자 검수결과

| 위치 | 현재 Markdown | 원문 대조 결과 / 수정안 | 판정 |
|---|---|---|---|
| `2024_LU_official+solutions.md` 327행, 07번 (1) 해설 | `주변부 유권자들(060076「히 0679/의` | `주변부 유권자들(peripheral voters)의` | **수정 필요**. PDF 렌더링 이미지에서 확인한 OCR 오인식이다. |
| `2024_LU_official+solutions.md` 576행, 12번 (3) 해설 | `「소피스테스』` | `『소피스테스』` | **수정 필요**. 책명 괄호가 여는 꺾쇠와 닫는 겹낫표로 잘못 섞였다. PDF 원문은 `『소피스테스』`이다. |
| 전 문항 | `정답 : (n)` 항목 없음 | 각 문항 `문항 해설` 또는 `문항구분` 아래에 원문 PDF의 정답 번호를 `정답 : (n)` 형식으로 추가 | **구조적 누락**. 2025ㆍ2026 Markdown과의 형식 통일성 측면에서도 보완 필요하다. |

## 이미지(그래프 등) 검수결과

- Markdown 내부 이미지 참조는 없다.
- 원문 PDF의 해설 본문에 반드시 별도 이미지로 추출해야 할 해설용 그림은 확인되지 않았다.
- 원문 PDF에 보이는 도표ㆍ그림 중 상당수는 문제 원문 또는 제시문 영역에 속하므로, 문제 원문 제외 지시에 따라 Markdown에 포함하지 않는 처리가 타당하다.

## 표 검수결과

- 해설 본문에 별도 표 이미지로 보존해야 하는 표는 확인되지 않았다.
- 수식 또는 수치 표현은 대부분 인라인 텍스트로 옮겨져 있다. 다만 2024 파일은 OCR 기반 흔적이 일부 남아 있으므로 `k`, `ℓ`, `DC_비`, `OC_맑음` 등 수식형 표현은 GitHub 표시 환경에서 재확인하는 것이 좋다.

## 잘못되었다고 보여지는 부분 개선안

### PDF 렌더링/심층 OCR 확인으로 얻은 개선안

1. 07번 (1) 해설:
   - 현재: `주변부 유권자들(060076「히 0679/의`
   - 수정: `주변부 유권자들(peripheral voters)의`

2. 12번 (3) 해설:
   - 현재: `「소피스테스』에서 보이는 플라톤의 진리관`
   - 수정: `『소피스테스』에서 보이는 플라톤의 진리관`

3. 전 문항 정답 표시:
   - 현재: 정답 번호가 별도 항목으로 없음.
   - 수정: 원문 PDF의 각 문항 `정답 :` 번호를 확인하여 `정답 : (n)` 형식으로 추가.

### 추론상 적절하게 개선한 안

- `정답 해설` / `오답 해설` 라벨은 원문 PDF의 회색 라벨 정보를 Markdown 구조로 보존하려면 다음과 같이 분리하는 편이 좋다.

```markdown
### 정답 해설

(3) ...

### 오답 해설

(1) ...
(2) ...
...
```

현재처럼 `### 선택지별 해설` 하나로 통합해도 내용 전달은 가능하지만, 원문 구조 보존 목적에는 위 형식이 더 적합하다.

## 사용자가 직접 확인하였으면 하는 부분

- 2024년 파일은 정답 번호가 전부 빠져 있으므로, 최종 업로드 전 원문 PDF 기준으로 30문항 전체 정답 번호를 일괄 입력해야 한다.
- 2024년 파일에는 인용부호의 여닫이 형태가 혼재된 대목이 일부 있다. 의미가 바뀌는 수준은 위 표의 두 건이 핵심이지만, GitHub 업로드용으로는 따옴표와 책명 괄호를 일괄 정규화하는 것을 권장한다.

# 2025학년도

## 검토요약

- 원문 PDF: 82쪽 스캔본.
- Markdown: 문항별 해설 `01`부터 `30`까지 모두 존재한다.
- 각 문항에 `정답 : (n)` 항목이 있으며, 총 30개로 문항 수와 일치한다.
- OCR 흔적으로 보이는 명백한 문자열 깨짐은 발견되지 않았다.
- 원문 PDF의 문제 원문 영역에는 표ㆍ그림ㆍ그래프가 존재하지만, 해설 영역 내부에 별도 이미지로 추출해야 할 표ㆍ그림은 확인되지 않았다.

## 오탈자 검수결과

- 중대한 누락 글자나 명백한 OCR 깨짐은 발견되지 않았다.
- 문항번호 01-30, 문항구분, 평가 목표, 문항 해설, 선택지별 해설의 기본 구조가 모두 유지되어 있다.
- 원문 PDF의 `정답 해설` / `오답 해설` 회색 라벨은 Markdown에서 선택지별 설명으로 통합되어 있다. 이는 구조 변환상의 선택으로 보이며, 정답 번호가 별도 기재되어 있어 2024년 파일보다 정보 손실은 작다.

## 이미지(그래프 등) 검수결과

- Markdown 내부 이미지 참조는 없다.
- 원문 PDF의 그림ㆍ표 중 확인된 것은 주로 문제 원문 또는 <보기> 영역에 속한다. 문제 원문 제외 지시에 비추어 Markdown에 포함하지 않은 것은 적절하다.
- 해설 본문에 독립 이미지로 보존해야 할 누락 그래프는 발견되지 않았다.

## 표 검수결과

- 해설 본문 안에서 표 자체를 별도 이미지로 추출해야 하는 사례는 발견되지 않았다.
- 문제 원문 <보기>의 표들은 해설에서 텍스트로만 언급된다. 원문 문제를 제외하는 방침을 유지한다면 현재 방식이 타당하다.

## 잘못되었다고 보여지는 부분 개선안

### PDF 렌더링/심층 OCR 확인으로 얻은 개선안

- 필수 수정으로 볼 만한 명확한 오탈자는 발견되지 않았다.

### 추론상 적절하게 개선한 안

- 원문 구조 보존성을 더 높이려면 2025년 파일도 2024년 개선안과 같이 `정답 해설` / `오답 해설`을 별도 소제목으로 둘 수 있다. 다만 현재 파일은 정답 번호와 선택지별 해설이 모두 있어 필수 오류로 보기는 어렵다.

## 사용자가 직접 확인하였으면 하는 부분

- 문제 원문 <보기>에 포함된 표ㆍ그림을 완전히 배제하는 정책을 계속 유지할지 확인하면 좋다. 해설만 단독으로 분석할 때 일부 선택지 해설이 `그림`, `표`, `<보기>`를 참조하므로, 별도 문제 원문 데이터셋과 연결되는 구조라면 현재 방식이 적절하다.
