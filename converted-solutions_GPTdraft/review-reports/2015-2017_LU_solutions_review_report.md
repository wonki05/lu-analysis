# LEET 언어이해 공식 해설 Markdown 검수 보고서 (2015-2017)

## 0. 검수 범위와 기준

- 대상 연도: 2015, 2016, 2017
- 대상 과목: 언어이해(LU)
- 원문: `법전협_[연도] LEET 해설_언어이해.pdf`
- 검수 대상: `[연도]_LU_official+solutions.md`
- 검수 기준: 사용자가 명시한 대로 문제 원문, 제시문, 보기 자체는 Markdown에 포함하지 않는 것을 정상으로 보았다. 따라서 문제/제시문/보기 안에 있는 표, 그림, 그래프는 해설 본문에 해당하지 않는 한 누락으로 판정하지 않았다.

### 검수 방법

1. 원문 PDF 3개를 전부 200dpi 이미지로 렌더링하여 페이지 단위로 확인했다.
   - 2015: 78쪽
   - 2016: 84쪽
   - 2017: 85쪽
2. 각 Markdown 파일의 문항 수와 문항 번호 배열을 확인했다.
   - 2015, 2016, 2017 모두 `## 01`부터 `## 35`까지 존재하고 순서 이상은 없었다.
3. 이미지/표/수식 가능성이 있는 페이지는 렌더링 이미지를 직접 확인했다. PDF가 이미지 기반이라 자동 OCR은 한글 띄어쓰기, 한자, 영문·기호에서 불안정했으므로, 확정 지적은 원문 렌더링 이미지를 직접 판독한 경우로 한정했다.
4. 원문에서 반복되는 파란색/회색 라벨 박스(문항 성격, 평가 목표, 문제 풀이, 정답 해설, 오답 해설 등)는 Markdown의 구조화된 제목과 목록으로 변환된 것으로 보았다. 다만 “원문 완전 전사” 기준에서는 라벨명 자체의 변환 여부를 별도 확인할 필요가 있다.

## 1. 2015학년도

### 1.1 검토요약

- 문항 번호: `01`~`35` 모두 존재한다.
- 해설 본문 중 **확정적으로 수정이 필요한 부분**은 2015년 02번 해설의 표 누락 1건이다.
- 출제방향 부분의 소제목에서 원문과 다른 띄어쓰기/조사가 확인된다. 본문 의미에는 영향이 없지만 원문 전사 기준에서는 수정 권장이다.
- 해설 내 별도 이미지 누락은 확인되지 않았다. 원문에서 보이는 그림·도표 중 다수는 문제/보기/제시문 영역에 있어, 문제 원문 제외 지시에 따라 Markdown에 포함하지 않는 것이 맞다.

### 1.2 오탈자 검수결과

#### 확정 수정 권장: 출제방향 소제목 표기

- 현재 Markdown: `## 1. 출제의 기본방향`
- 원문 PDF: `1. 출제의 기본 방향`
- 개선안: `## 1. 출제의 기본 방향`
- 판정 근거: 원문 렌더링 직접 판독

- 현재 Markdown: `## 5. 문항 출제시 유의점 및 강조점`
- 원문 PDF: `5. 문항 출제 시의 유의점 및 강조점`
- 개선안: `## 5. 문항 출제 시의 유의점 및 강조점`
- 판정 근거: 원문 렌더링 직접 판독

#### 참고: 최상위 제목

- 현재 Markdown: `# 출제방향`
- 원문 PDF 표제: `2015학년도 언어이해 영역 출제 방향`
- 판단: 저장소 전체 규칙에 맞춘 축약 제목이라면 유지 가능하다. 원문 완전 전사 기준이라면 `# 2015학년도 언어이해 영역 출제 방향`으로 바꾸는 것이 더 엄밀하다.

### 1.3 이미지(그래프 등) 검수결과

- 해설 본문 자체에 별도 이미지가 포함되어야 하는 사례는 확인되지 않았다.
- 원문에 보이는 그림·보기 박스·문제 표는 문제 원문 영역에 해당하는 것이어서, 이번 지시에 따라 제외된 것으로 판단했다.
- 사용자 확인 권장: 향후 Markdown만으로 해설을 독립적으로 읽게 만들 계획이라면, 해설이 `<보기>`나 `<그림>`을 강하게 참조하는 문항에 한해 문제 원문 일부를 별도 이미지로 첨부할지 정책 판단이 필요하다.

### 1.4 표 검수결과

#### 확정 오류: 02번 해설의 표에서 1개 행 누락

2015년 02번 해설 중 예와 형을 대비하는 표에서 원문 PDF에는 다음 행이 존재한다.

```markdown
| 정문 | 사형 |
```

그러나 현재 Markdown 표에는 해당 행이 없다.

현재 Markdown 표:

```markdown
| 예 | 형 |
|---|---|
| 무질서를 막는 것 | 무질서를 막는 것 |
| 의를 좇는 이가 나아갈 바 | 해를 피하려는 이가 설 곳 |
| 도리를 밝혀 상벌을 정하도록 한 것 | 사실에 터 잡아 시비를 가리도록 한 것 |
```

개선안:

```markdown
| 예 | 형 |
|---|---|
| 무질서를 막는 것 | 무질서를 막는 것 |
| 정문 | 사형 |
| 의를 좇는 이가 나아갈 바 | 해를 피하려는 이가 설 곳 |
| 도리를 밝혀 상벌을 정하도록 한 것 | 사실에 터 잡아 시비를 가리도록 한 것 |
```

- 판정 근거: 원문 렌더링 직접 판독
- 성격: 표 내용 누락. OCR 추론이 아니라 원문 이미지에서 직접 확인한 수정안이다.

## 4. 공통 확인 사항

### 4.1 문항 범위와 문항번호

- 2015, 2016, 2017 Markdown 모두 `01`부터 `35`까지 문항이 존재한다.
- 순서상 누락 또는 중복 문항은 확인되지 않았다.
- 문제 원문 범위 표기, 예컨대 `[01~03] 다음 글을 읽고 물음에 답하시오.` 같은 문구는 문제 원문에 해당하므로, 현재 Markdown에 없는 것을 오류로 보지 않았다.

### 4.2 강조표시 및 라벨

- 원문 PDF의 파란색/회색 라벨 박스는 Markdown에서 `### 문항구분`, `### 문항 해설`, `### 선택지별 해설`, `### <보기> 해설` 등으로 구조화되어 있다.
- 본문 내용 보존 관점에서는 큰 문제는 확인되지 않았다.
- 단, 원문 라벨 자체까지 완전 전사하려는 정책이라면 다음 변환은 사용자 확인이 필요하다.
  - 원문 `문제 풀이` → Markdown `문항 해설`
  - 원문 `정답 해설`, `오답 해설` → Markdown `선택지별 해설` 아래 선택지별 문단
  - 원문 `<보기> 해설`은 대체로 Markdown에서도 별도 heading으로 보존됨

### 4.3 이미지/표 정책

- 현재 검수대상 파일은 `*_official+solutions.md` 단일 파일이며, 이미지 패키지 ZIP이 아니다.
- 해설 본문에 필수로 들어가야 할 이미지 누락은 확인되지 않았다.
- 다만 문제/보기/제시문에 포함된 도표와 그림은 해설이 참조하는 경우가 있으므로, 저장소 사용 목적이 “해설만 독립적으로 읽히는 자료”라면 별도 정책 판단이 필요하다.

### 4.4 사용자 직접 확인 권장 사항

1. 최상위 제목을 현재처럼 `# 출제방향`으로 둘지, 원문 표제처럼 `[연도]학년도 언어이해 영역 출제 방향`으로 둘지 결정할 필요가 있다.
2. 원문 라벨명과 Markdown 섹션명을 완전히 일치시킬지, 현재처럼 AI 분석 친화적인 구조화 제목으로 둘지 결정할 필요가 있다.
3. 수식의 하첨자·위첨자를 plain text(`E_1`, `2^n`)로 둘지, Markdown 수식(`$E_1$`, `$2^n$`)으로 바꿀지 결정할 필요가 있다.
4. 2015년 02번 표의 `정문 | 사형` 행은 내용 누락이므로 우선 수정하는 것이 좋다.
