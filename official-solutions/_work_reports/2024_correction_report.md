직선 인용부호(큰따옴표 334, 작은따옴표 250)를 위치 규칙으로 둥근 인용부호로 전량 변환. 단일/이중 어긋난 쌍 12건 교정(`‘…”`→`“…”` 등), 방향 어긋난 둥근 따옴표 4건 교정(예: 268행 `”k-익명성` → `“k-익명성`). 하드랩으로 인한 행두 인용부호 12곳은 전부 여는 부호로 정상 판정됨을 개별 확인.

### 5. 수식·단위 표기 확인 (검수보고서 지적, 정상)
검수보고서가 재확인 요청한 표기를 PDF와 대조 — 모두 정상 렌더링 확인:
- `k-익명성`, `ℓ-다양성`(ℓ=U+2113), `k값`, `k=2/3/4`, `ℓ=2`, `4/6=66.67%` — PDF 일치
- `DC_비`, `DC_맑음`, `OC_비`, `OC_맑음`, `DC_비-DC_맑음>0` — 아래첨자를 밑줄 인라인으로 표기(허용 가능)
- `0.05μs`, `20nm` — 표준 단위 표기(PDF의 ㎲·㎚에 대응)

## 유지 판단 사항
- **초안 하드랩(줄 단위 줄바꿈) 유지**: 초안은 원문 줄 단위로 하드랩되어 있어(약 610개 줄바꿈) 2025·2026의 문단=한 줄 형식과 다르다. 그러나 (1) 검수보고서가 이를 결함으로 지적하지 않았고, (2) 한글 어절 경계/어절 내부 판정 없이 재접합(reflow)하면 접합 공백을 오삽입(`점에|서”`→`점에서”` vs `반증가능성을|학문의`→띄어쓰기 필요)할 위험이 커 안전한 자동 재접합이 불가하므로, 원문 내용 보존을 위해 하드랩을 유지. 필요 시 재-OCR 기반 재생성으로 문단화 권장.
- **`정답 해설`/`오답 해설` 라벨 미추가**: 검수보고서가 선택적 개선안으로 제시(정답 번호 보존됨). 기존 연도와 동일하게 `### 선택지별 해설` 구조 유지(전역 구조 개편 금지 원칙).
- **문항구분 `* 문항 성격 : 문항유형 : X / 내용영역 : Y` 형식**: 값·라벨 모두 PDF와 일치하여 유지.
