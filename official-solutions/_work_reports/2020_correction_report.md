### 2. 문항구분 `문항유형 :` 라벨 복원 (30건)
PDF 해설 박스는 `문항유형 : …` 행이나 초안은 이를 `* 문항 성격 : …`으로 표기. 원문 라벨로 복원:
- `* 문항 성격 : [값]` → `* 문항유형 : [값]` (01~30 전 문항). 값 자체는 원문과 일치하여 유지.

### 3. `내용영역 :` 행 신규 추가 (30건, 전 문항 PDF 판독)
