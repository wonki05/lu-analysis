# LEET 언어이해 해설 Markdown 교정 — 최종 종합 보고서 (2012–2026)

## 1. 프로젝트 개요

- **목적**: 공식 LEET 언어이해(LU) 해설 PDF(법전협)에서 변환된 Markdown 초안 15개 연도분(2012~2026)을 원본 PDF와 대조하여 검수·교정하고, 연도별 작업보고서와 최종 검증 문서를 산출한다.
- **입력(읽기 전용, 불변)**: `converted-solutions_GPTdraft/`(MD 초안), `converted-solutions_GPTdraft/review-reports/`(검수보고서), `docs_solutions/*.pdf`(원본, 스캔 이미지 — 텍스트 레이어 없음).
- **산출물(편집 대상)**: `official-solutions/[연도]_solution_files/[연도]_LU_official+solutions.md` + 필요한 이미지, 그리고 `official-solutions/_work_reports/`의 연도별 보고서·검증 문서.

### 작업 원칙(전 연도 공통 적용)
1. 입력 파일 절대 불변, 산출물 사본만 편집.
2. 문제·지문·선택지·<보기> 원문을 신규 추가하지 않음(단, 해설의 표·도식·이미지·캡션은 보존).
3. 헤딩 구조 전역 개편 금지 — 검수보고서 지적 **또는** PDF 확인이 있는 개별 항목만 정정.
4. **PDF 렌더링이 검수보고서보다 우선**(상충 시 PDF 채택).
5. 선택지 번호 (1)~(5) 통일, 본문에 작업자 메모·불확실 표기 금지(보고서에만 기록).
6. 원문에 없는 내용 무생성. 미해결은 (a) 원본 PDF 자체 불완전, (b) PDF로 확인 불가, (c) 두 출처 상충 시에만 보고서에 한계로 명시.

- **연도별 작업보고서 15개**: `[연도]_correction_report.md`.

## 3. 공통 수정 유형과 연도별 적용

### (A) 출제방향 구조·표기 정정 — **전 15개 연도 적용**
- 초안이 타 연도 양식으로 재구성해 둔 섹션 제목/순서를 각 연도 PDF 원문에 맞춰 정정.
- 연도별 상이 확인: 섹션 수(2012=3, 2021·2022=4, 그 외=5), 3번 제목("제시문 및 문항" ↔ "문항 구성"), 5번/병합 제목("문항 출제 시의 유의점 및 강조점" ↔ "출제 시 유의점"), 1번("기본방향"→"기본 방향"). 상세는 `validation_summary.md` §3.

### (B) 정답 행 추가·정정

### (F) 문항구분 형식 전 연도 통일 (사후 사용자 지시, 2026-07-04)

- **2013 한정**: 15개 문항은 원문 값이 내용영역만 존재(`인문ㆍ예술`, `사회`, `인문`)하여 `* 내용영역 :`만 생성(문항유형 무생성 — 원문에 없는 내용 무생성 원칙).

### (G) 2012 33~35번 해설 보완 (사후 사용자 제공 발췌 PDF, 2026-07-06)
- 최초 원본 PDF가 33번 오답 해설 (3)에서 종료되어 미해결로 남겼던 부분을, 사용자 제공 발췌 공식 해설서 PDF(`법전협_2012 LEET 해설_언어이해_33-35.pdf`, 도서 330~337쪽)를 고배율 판독하여 전량 보완.
- 보완 내역: 33번 선택지별 해설 (4)·(5) 추가(기존 (1)~(3)은 발췌본 대조로 완전함 확인), 34번 전체 추가(정답 (5)), 35번 전체 추가(정답 (3)) — 문항구분·문항 해설·선택지별 해설을 기존 파일 서식(문항유형/내용영역/평가 목표/정답 bullet, 번호순 선택지 해설)에 맞춰 전사.
- 검증: 2012 문항 35 = 정답 행 35, 인용부호 균형(겹 194/194, 홑 185/185), 직선 따옴표 0. 이로써 §5의 한계 1(2012 원본 PDF 불완전)은 해소됨.

### (H) 정답 표기 위치 전 연도 통일 (사후 사용자 지시, 2026-07-06)
- 사용자 지시에 따라 `### 문항 해설` 본문 안의 평문 `정답 : (n)` 행을 **`### 문항구분`의 마지막 bullet(`* 정답 : (n)`)**으로 이동하여 2012 형식으로 전 연도 통일. 형식은 기존 `(n)` 괄호 숫자 유지.
- 대상: 2015·2016·2018(각 35건), 2020·2022·2023·2024·2025·2026(각 30건) — 총 275건 이동. 2012·2013·2014·2017·2019·2021은 이미 bullet 형식이라 무변경.
- **정답 값 보존 검증**: 이동 전/후 문항별 정답 맵 전건 동일(9개 연도), 2023~2026은 본 보고서·연도별 보고서의 정답표와 재대조하여 전건 일치. 최종적으로 15개 연도 전건 `문항 수 = 정답 bullet 수`, 평문 정답 행 0건, 위치(문항구분 내) 전건 확인.
- 부수 정정: (1) 2015는 작업 시점에 파일이 연속 bullet 스타일(외부 포매터 반영)로 바뀌어 있어 정답 bullet도 연속 스타일로 정렬(빈 줄 삽입분 제거, 35건). (2) 2023 문항 20·25·29의 평가 목표 bullet 내부에 남아 있던 OCR 문장 분단 3건(`묻고∥있다`, `이해하고∥있는지`, `해석하∥고`)을 병합 — 기존 §(C) 2023 문장 병합 작업의 누락분.

## 5. 한계 및 권고

1. ~~**2012 원본 PDF 불완전**~~ **(해소, 2026-07-06)**: 33번 오답 해설 (3) 이후(33번 (4)·(5), 34·35번)가 최초 원본에 부재했으나, 사용자 제공 발췌 PDF로 전량 보완 완료 — §3(G) 참조.
2. **지문 인용 부호 소수 잔여 불균형**(2016·2018·2019·2021 각 1~3건): OCR이 지문 인용의 여는/닫는 부호를 누락한 경계 표기 문제. 정답·의미 무영향이며 `validation_summary.md` §4에 파일 행·문항·선택지까지 특정. 위치 확정에 PDF 판독이 필요해(임의 배치 시 오정정 위험) 본 작업에서는 정확 위치 명시로 처리.
3. **2023·2024 초안 OCR 손상**: 인용부호·조사/어절 내부 공백 잔여가 광범위. 의미·정답은 보존되나 완전한 표기 정합을 위해 **원본 PDF 재-OCR 후 재생성**을 권장(각 연도 보고서 명시).
4. **2024 하드랩 유지**: 초안이 원문 줄 단위 하드랩(약 610개). 한글 어절 경계 판정 없이 재접합 시 공백 오삽입 위험이 커 원문 보존을 위해 유지. 재-OCR 기반 문단화 권장.
5. **구조 표준화 보류**: `정답 해설`/`오답 해설` 색상 라벨은 초안의 `### 선택지별 해설` 구조로 통일되어 있음(전역 구조 개편 금지 원칙). 정답 번호는 전건 보존.

## 6. 결론

2012~2026 15개 연도 언어이해 해설 MD의 **필수 정합 항목(정답 완전성, 직선 따옴표 제거, 출제방향 구조, 이미지 링크·캡션)을 전건 충족**했으며, 각 수정은 원본 PDF 대조로 검증했다. 잔여 항목은 소수의 지문 인용 부호 불균형(위치 특정, 무영향)과 초안 OCR 손상이 큰 2023·2024의 재-OCR 권장 사항으로 한정된다. 세부 근거는 연도별 `[연도]_correction_report.md`와 `validation_summary.md`에 기록되어 있다.
