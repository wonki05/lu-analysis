### 2. 문항구분 `문항유형 :` 라벨 복원 (30건, 전 문항)
PDF 원문 해설 박스는 `문항 성격` 헤더 아래에 `문항유형 : …`와 `내용영역 : …` 두 행으로 구성되나, 초안은 `문항유형` 라벨을 누락하고 그 값을 `* 문항 성격 : …`으로 표기했다. 원문 라벨을 복원:
- `* 문항 성격 : [값]` → `* 문항유형 : [값]` (01~30 전 문항)
- 라벨만 복원했으며 값 자체는 초안 그대로 유지(값이 원문과 일치함을 스팟 대조로 확인: 01=주제,구조,관점 파악 / 14=정보의 확인과 재구성 / 28=주제,구조,관점 파악 / 29=정보의 평가와 적용 / 30=정보의 추론과 해석).
- `내용영역`은 이미 원문과 동일한 라벨/표기로 존재하여 그대로 유지.
