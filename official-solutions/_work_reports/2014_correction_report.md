## 판단 보류 사항 (기존 구조 유지)
- 2014 원문 해설 박스는 `문항 성격` 라벨 아래 `문항유형 : … / 내용영역 : …` 구조인데, 초안은 `* 문항 성격 : [문항유형값]` + `* 내용 영역 : [값]`으로 옮겨 `문항유형 :` 라벨이 없고 `내용영역`을 `내용 영역`으로 띄어 씀. 값 정보는 모두 보존됨.
  - 지침은 `문항유형 :` 라벨 복원을 2019·2020에만 명시했으므로, 2014는 기존 구조를 유지함(전역 라벨 변경은 구조 변경에 해당). 사용자가 전 연도 통일을 원할 경우 별도 일괄 작업 필요.
