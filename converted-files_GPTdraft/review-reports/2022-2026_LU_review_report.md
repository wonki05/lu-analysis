# LEET 언어이해 Markdown 변환물 검수결과 보고서

## 1. 검수 개요

- **작업대상**: 2022~2026학년도 LEET 언어이해 문제지 및 Markdown 변환물
- **원문 비교대상**: `[연도]_LEET_언어이해_문제지(홀수형).pdf`
- **검수대상**: `[연도]_LU_improved_outputs.zip`
- **검수 목적**: 원문 PDF의 문항, 제시문, 선택지, 표, 수식, 이미지, 강조표시가 Markdown 변환물에 누락 없이 반영되었는지 확인
- **검수 기준**:
  - 누락된 글자 여부
  - 누락된 수식 여부
  - 누락된 표·이미지 여부
  - 누락된 강조표시(볼드, 밑줄 등) 여부
  - 제시문의 문항범위 및 문항번호 기재 정확성
  - 기타 Markdown 변환상 문제 여부

## 2. 연도별 검수결과

## 2022년 LU

- **문항 구조**
  - `[01-03]`부터 `[28-30]`까지 10개 제시문 묶음이 모두 존재합니다.
  - `## 01`부터 `## 30`까지 30문항이 빠짐없이 구성되어 있습니다.

- **선택지 구조**
  - 1~30번 모든 문항에 `(1)`~`(5)` 선택지 5개가 존재합니다.

- **이미지 파일**
  - 원문상 필요한 시각 자료 3개가 ZIP 내부에 존재합니다.
  - Markdown에서도 해당 이미지가 모두 참조됩니다.
  - 확인된 이미지 파일:
    - `2022_LU_images_11_1.png`
    - `2022_LU_images_text_16-18_1.png`
    - `2022_LU_images_text_16-18_2.png`

- **이미지 누락 여부**
  - 11번의 망막/신호 그래프가 이미지로 보존되어 있습니다.
  - 16~18번의 클러스터링 관련 그림 2개가 이미지로 보존되어 있습니다.
  - 명백한 이미지 누락은 확인되지 않았습니다.

- **표·수식 누락 여부**
  - 별도 표나 복잡 수식이 이미지로 빠져야 할 부분은 확인되지 않았습니다.
  - 도식은 PNG 파일로 보존되어 있습니다.

- **확인된 문제점**
  - 원문 PDF에서 부정형 문항의 핵심어인 **“않은/않는”** 등에 밑줄이 표시되어 있으나, Markdown의 다수 문항 제목에서는 `<u>...</u>` 처리가 빠져 있습니다.
  - 예: 원문 2번 문항의 `적절하지 않은 것은?`에서 **“않은”**에 밑줄이 있으나, Markdown에는 `적절하지 않은 것은?`으로만 기재되어 있습니다.

- **판정**
  - 문항 구조, 선택지 구조, 이미지 보존은 양호합니다.
  - 다만 **문항 stem의 밑줄 강조 누락이 반복적으로 존재**하므로 최종 업로드 전 수정이 필요합니다.

## 2025년 LU

- **문항 구조**
  - 원문 PDF 상단의 문제유형은 **홀수형**으로 확인됩니다.
  - Markdown에는 `[01-03]`부터 `[28-30]`까지 10개 제시문 묶음이 모두 존재합니다.
  - `## 01`부터 `## 30`까지 30문항이 빠짐없이 구성되어 있습니다.

- **선택지 구조**
  - 1~30번 모든 문항에 `(1)`~`(5)` 선택지 5개가 존재합니다.
  - 홀수형 PDF 기준 선택지 배열과 Markdown 선택지 배열이 일치합니다.

- **이미지 파일**
  - ZIP 내부의 3개 이미지가 모두 Markdown에서 정상 참조됩니다.
  - 확인된 이미지 파일:
    - `2025_LU_images_12_1.png`
    - `2025_LU_images_text_16-18_1.png`
    - `2025_LU_images_27_1.png`

- **이미지 누락 여부**
  - 12번 `<보기>`의 선거법 개정 관련 여론 및 국기법 개정 관련 여론 그래프가 이미지로 보존되어 있습니다.
  - 16~18번 공통 제시문의 솔로우 성장모형 그래프가 이미지로 보존되어 있습니다.
  - 27번 `<보기>`의 트랜잭션 실행 순서 및 데이터베이스 상태 변화 도식이 이미지로 보존되어 있습니다.
  - 명백한 이미지 누락은 확인되지 않았습니다.

- **표·수식 누락 여부**
  - 6번 `<보기>`의 효소·기질·산물·포르피린증 표는 Markdown 표로 반영되어 있습니다.
  - 16~18번 지문의 변수 `y`, `k`, `c`, `i`, `s`, `d`와 수식 `\Delta k = i - dk`가 LaTeX 형식으로 반영되어 있습니다.
  - 16~18번의 `[A]` 표지는 해당 단락 첫머리에 기재되어 있어 문항 참조에는 문제가 없습니다.

- **강조표시**
  - 문항 stem의 부정어 밑줄은 `일치하지 <u>않는</u>`, `적절하지 <u>않은</u>`과 같이 HTML 밑줄 태그로 반영되어 있습니다.
  - 10~12번 공통 제시문의 `네 가지 모델`, 28~30번 공통 제시문의 `심포니 모델`, `시네마 모델`, `조각 모델`은 `<kbd>...</kbd>`로 처리되어 원문의 박스형 표시를 어느 정도 보존하고 있습니다.
  - 다만 원문 지문에서 밑줄 표시가 있는 참조 대상 어구 중 일부가 Markdown에서는 일반 텍스트로 처리되어 있습니다.
  - 확인된 예:
    - 1~3번 공통 제시문의 `㉠ 피에 굶주린 형법전`, `㉡ 자연의 제일법칙`, `㉢ 교수대에 낭비된 감수성`, `㉣ 문학적 공범자`, `㉤ 봉쇄된 정의`
    - 19~21번 공통 제시문의 `㉠ 엄격한 기준`
    - 25~27번 공통 제시문의 `㉠ 트랜잭션의 동시성 제어`

- **확인된 문제점**
  - 문항·선택지·이미지·표·수식의 누락은 확인되지 않았습니다.
  - 다만 지문 내 참조 대상 어구의 밑줄 강조가 일부 누락되어 있으므로, 원문 강조 보존 기준상 해당 어구에는 `<u>...</u>` 처리를 추가하는 것이 필요합니다.

- **판정**
  - 문항 구조, 선택지 구조, 이미지 보존, 표·수식 반영은 전반적으로 양호합니다.
  - 최종 업로드 전에는 **지문 내 참조 대상 어구의 밑줄 강조 누락**을 수정하는 것이 필요합니다.

## 3. 사용자가 직접 확인했으면 하는 부분

- **2025년 지문 내 참조 대상 어구의 밑줄 처리 범위**
  - 2025년은 문항 stem의 부정어 밑줄은 반영되어 있으나, 지문 안의 참조 대상 어구 밑줄은 일부 누락되어 있습니다.
  - 특히 1~3번의 `㉠`~`㉤`, 19~21번의 `㉠`, 25~27번의 `㉠`에 딸린 어구를 원문처럼 밑줄 처리할지 직접 확인하는 것이 좋습니다.

- **2022~2024, 2026년의 문자 단위 최종 확인**
  - 이 네 연도 PDF는 텍스트 추출이 되지 않거나 매우 제한적입니다.
  - 모든 글자를 자동으로 원문 텍스트와 완전 대조하기 어렵습니다.
  - 렌더링 이미지 기준으로 구조와 주요 요소를 확인했고 명백한 문제를 위에 적었지만, 최종본 확정 전에는 특히 오탈자와 부정형 문항 stem을 한 번 더 사람이 확인하는 것이 안전합니다.

- **부정형 문항 밑줄 일괄 수정**
  - 2022, 2023, 2024, 2026에서 `적절하지 않은`, `일치하지 않는` 등의 `않은/않는` 밑줄이 Markdown에 빠진 사례가 반복됩니다.
  - 원문 강조 보존 원칙상 전수 수정 대상입니다.

- **이미지 alt text 보강**
  - 모든 이미지 링크가 `![](파일명)`처럼 빈 alt text입니다.
  - 이미지 파일 자체는 존재하지만, GitHub/AI 분석 목적이라면 `![11번 망막 신호 그래프](...)`처럼 간단한 설명을 넣는 편이 더 적합합니다.

## 4. 종합 판정

- 2022~2026년 전체에 대해 문항 수와 선택지 수는 전반적으로 정상입니다.
- 이미지 파일은 각 연도별로 ZIP 내부에 존재하며, Markdown에서도 대체로 정상 참조됩니다.
- 2025년은 문항 구조, 선택지 구조, 이미지 보존, 표·수식 반영이 전반적으로 양호합니다.
- 2025년은 지문 내 참조 대상 어구의 밑줄 강조가 일부 누락되어 있어, 해당 부분은 최종 업로드 전 보완이 필요합니다.
- 2022, 2023, 2024, 2026년은 공통적으로 **부정형 문항의 밑줄 강조 누락**이 반복됩니다.
- 2024년은 개인정보 마스킹 별표(`**`)의 Markdown 문법 충돌 가능성이 있습니다.
- 2026년은 19~21번 지문에 `수호성년데` → `수호성녀던데`로 수정해야 할 확정 오탈자가 있습니다.
