# LEET 언어이해 문항분류지침서 (개정판)

작성 목적: 2012~2026학년도 LEET 언어이해 기출문항을 **선택지의 정오가 갈리는 논리적 구성방식**에 따라 분류하기 위한 지침서이다. 후속 AI 분류 작업자가 이 문서만 읽고도 각 유형이 "선택지의 어떤 논리 조작을 잡아내는 유형인지"를 판별할 수 있도록 설계하였다.

사용 대상: 후속 AI 분류 작업자(주 사용자) 및 사람 검토자.

기본 원칙
- 배경지식이 아니라 **제시문 및 <보기>에 주어진 정보**만을 근거로 판단한다(논리학 지식은 예외적으로 허용).
- 분류의 단위는 문항이 아니라 **선택지(또는 <보기>의 ㄱ·ㄴ·ㄷ)**이다.
- 공식해설의 정답은 신뢰하되, 해설의 '문항유형' 라벨은 분류 기준으로 사용하지 않는다.
- 하나의 마지막층위는 오직 하나의 중간층위에만 속한다(교차 배치 금지).

이 개정판이 초안(가안)과 다른 점(요약)
1. 대분류 10개는 그대로 유지하되, **중간층위 이름을 더 직관적으로 다듬고** 마지막층위를 50개 → **48개**로 정리하였다.
2. 초안에서 서로 다른 대분류에 흩어져 사실상 같은 논리를 잡던 마지막층위들(예: `단계산물혼동`/`입출력방향오류`/`모델단계오류`, `사례역전`/`실질반례오인`/`보충근거오판`, `필자평가혼동`/`수용효과오판`)을 **"무엇을 대상으로 추론하는가"라는 단일 기준으로 재배치**하여 중복을 제거하였다(§0.4 동점 처리 원칙).
3. 초안의 `출제비중 …%`는 **측정치가 아니라 추정치**였다(초안 §13.2도 이를 인정). 개정판은 허위 정밀도를 피하기 위해 이를 **정성적 출제빈도 band(매우높음/높음/중간/낮음)**로 대체하고, **대표기출은 원문에서 직접 확인한 것만** 실었다.

## 0. 적용 범위와 태깅 원칙

### 0.1 적용 범위
- 대상 연도: 2012~2026학년도(15개년). 세트 구조는 2019학년도부터 "3문항 세트 × 10세트 = 30문항"으로 고정된다(그 이전은 35문항).
- 대상 문항: 언어이해 독해형 문항. 단, **2012·2013학년도 초반부의 한국어 어휘·어법 문항(각 연도 1~3번대)은 제외**한다.
- 참고 자료: `problem-set/{연도}_files/{연도}_LU_problems.md`, `official-solutions/{연도}_solution_files/{연도}_LU_official+solutions.md`.

### 0.2 세트의 전형적 구조(경험칙)
각 3문항 세트는 대체로 다음 순서를 따르며, 이는 자주 등장하는 유형을 예측하는 데 유용하다.
- **1번(확인형)**: "윗글의 내용과 일치하(지 않)는 것은?" → 주로 `#개념관계`, `#추론강도(미언급·반대확정)`.
- **2번(추론·설명형)**: "㉠에 대한 설명/추론", "A와 B에 대한 이해" → 주로 `#관점귀속`, `#조건관계`, `#층위관계`, `#인과판정`.
- **3번(적용·평가형)**: "<보기>를 평가/이해/적용" → 주로 `사례적용`, `강화약화`, `모형·자료·절차`.

단, 이는 예측 보조일 뿐이며 **최종 분류는 반드시 선택지별 판정 근거로 결정한다**(§0.4).

### 0.3 선택지 분해 관점
LEET 선택지는 대체로 `대상 귀속 + 관계 연산자 + 범위·조건 + 평가 기준`으로 이루어진다. 오답 선택지는 이 중 한 요소를 제시문 근거로부터 다음 여섯 방식 중 하나로 변형한다: **확대 / 축소 / 반전 / 누락 / 치환 / 과잉확정**. 분류자는 (1) 근거 문장을 찾고 (2) 어떤 변형이 일어났는지 지목한 뒤 (3) 태그를 붙인다.

### 0.4 동점 처리 원칙 (교차 중복을 없애는 핵심 규칙)
겉보기에 여러 유형에 해당할 수 있는 선택지는 **"선택지가 무엇을 대상으로 추론하는가"**로 귀속 대분류를 결정한다. 이 원칙이 초안의 최대 약점(같은 논리가 여러 곳에 흩어짐)을 해소한다.

- **"방향·단계·층위" 계열**(입력↔출력, 부분↔전체, 중간산물↔최종산물)
  - 대상이 **제시문 내부의 개념 층위**이면 → `4. 부분과 전체`
  - 대상이 **주어진 표·그림·의사결정표·수식**이면 → `8. 모형·자료·절차`
  - 대상이 **<보기>의 새로운 사례**이면 → `9. 사례적용`
- **"귀속·평가" 계열**(누구의 주장/평가인가)
  - 대상이 **논증·이론의 입장**이면 → `2. 이항대립`
  - 대상이 **문학·평론의 화자·서술자·독자·대중**이면 → `10. 문학·표현`
- **"지지·약화" 계열**
  - 자료·사례가 주장을 **강화/약화/반박하는지**를 평가하면 → `7. 강화약화`
  - 자료·사례를 본문 틀에 **정확히 대입·분류**하는지를 묻는 것이면 → `9. 사례적용`

### 0.5 중복 태깅 규칙
- 한 선택지는 주 태그 1개 + 보조 태그 0~2개를 가진다. 가장 직접적으로 정오를 가르는 조작이 주 태그이다.
- 마지막층위 이름은 대분류가 달라도 유일하다. 태그 표기는 `대분류 > #중간층위 > 마지막층위` 형식을 쓴다.

# 1. 논리구조

논리구조 유형은 선택지가 제시문 속 명제의 **조건·정의·규칙**을 형식적으로 변형하여 정오가 갈리는 경우이다. 전 영역에서 가장 기초적이고 반복적이며, 특히 규범·과학기술·의사결정 세트에서 밀도가 높다.

대표 세트: 2017학년도 카르네아데스의 널(범죄 성립 3요건), 2021학년도 프로세스 마이닝, 2024학년도 비식별화, 2026학년도 DMN. 출제빈도 **매우높음**.

## #조건관계

선택지가 결론 성립에 필요한 요건 또는 충분한 요건을 잘못 잡는 유형. 형식논리로 `P이면 Q`, `Q이려면 P`, `P·R이면 Q` 구조로 환원된다.

### 요건누락
결론 성립에 여러 조건이 함께 필요한데, 선택지가 그중 일부만으로 결론이 성립한다고 하는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2017학년도 2번·3번(범죄는 구성요건·위법성·유책성 중 하나만 빠져도 불성립), 2024학년도 30번(국제협약 적용 요건), 2026학년도 22번·23번(대안 가능성 원칙)
* 판정질문 : "이 선택지는 결론을 내기 위해 필요한 조건을 모두 통과했는가?"
* 예시 : 구성요건 해당성만으로 곧바로 범죄 성립을 말하면 위법성·유책성 요건을 누락한 것.

### 요건과잉
제시문이 요구하지 않은 조건을 선택지가 추가하여 결론 성립 범위를 부당하게 좁히는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2017학년도 2번(긴급피난 성립에 '상대방 행위의 위법성'을 과잉 요구 — 이는 정당방위의 요건), 2024학년도 29번, 2026학년도 6번
* 판정질문 : "제시문이 실제로 그 조건까지 요구했는가?"
* 예시 : 긴급피난은 위법한 침해에 대한 것일 필요가 없는데 정당방위처럼 위법성을 요구하면 요건과잉.

### 조건역전
필요조건과 충분조건, 전건과 후건의 방향을 뒤집는 오류. (원인/결과의 방향 뒤집기는 `5.#인과판정 > 인과방향전도`로 보낸다 — 형식 조건문인지 인과 서술인지로 구별.)
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2024학년도 5번(k값-잔존율의 조건 방향), 2026학년도 5번(적중정책 조건), 2013학년도 8번
* 판정질문 : "지문은 A가 B의 조건이라 했는가, B가 A의 조건이라 했는가?"
* 예시 : "조건 열을 모두 만족하면 규칙이 만족된다"를 "규칙이 만족되면 조건 열을 만족한다"로 뒤집으면 조건역전.

## #개념관계

선택지가 지문에 등장한 개념어를 그대로 쓰면서도 그 정의·외연·인접 개념과의 구분을 바꾸는 유형. 세트 1번(확인형)의 핵심 유형.

### 정의오인
지문이 명시한 정의 자체를 다르게 이해하거나 정의의 핵심 요소를 빠뜨리는 오류.
* 중요도 : 최상
* 출제빈도 : 매우높음
* 대표기출 : 2021학년도 1번(이벤트 로그의 정의), 2024학년도 4번(식별자/준식별자), 2026학년도 1번(권리 주체), 2025학년도 25번(트랜잭션)
* 판정질문 : "선택지가 쓰는 개념어가 지문 정의와 같은가?"
* 예시 : 이벤트 로그·식별자·민감속성·레코드 등은 지문 내부 정의로만 판정.

### 외연변조
개념의 적용 범위를 넓히거나 좁히거나, 특정 대상이 그 개념에 포함되는지 여부를 바꾸는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2024학년도 4번(민감속성은 비식별 처리 대상이 아님을 뒤집음), 2024학년도 26번, 2026학년도 2번
* 판정질문 : "이 대상은 지문상 그 개념의 외연 안에 들어가는가?"
* 예시 : 식별자·준식별자에만 수행하는 처리를 민감속성까지 확장하면 외연변조.

### 유사개념치환
서로 가까워 보이는 두 **개념어**를 같은 것으로 처리하는 오류. (두 **입장·이론**의 차이를 지우는 것은 `2.#비교대조 > 차이점삭제`.)
* 중요도 : 상
* 출제빈도 : 높음
* 대표기출 : 2021학년도 1번(프로세스 '수정' vs '확장'), 2024학년도 4번, 2025학년도 14번(수의주의 vs 불수의주의)
* 판정질문 : "지문은 두 개념을 구분했는가, 동일시했는가?"
* 예시 : 이벤트 로그와 프로세스 모델, 수정과 확장은 서로 바꿔 쓸 수 없다.

## #규칙관계

원칙·예외·기본값·특수규칙의 적용 범위를 선택지가 바꾸는 유형. 법·의사결정 세트 특화.

### 원칙예외전도
예외를 원칙처럼, 원칙을 예외처럼 말하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2017학년도 3번(위법성은 원칙, 위법성 조각은 예외), 2024학년도 30번, 2026학년도 5번
* 판정질문 : "지문상 이것은 일반규칙인가, 예외조건인가?"
* 예시 : 반환 예외 사유가 인정될 수 있다는 것과 반환청구가 반드시 배척된다는 것은 다르다.

### 기본특칙오인
별도 지정이 없을 때 적용되는 기본값을 다른 값으로 바꾸거나, 특정 조건에서만 끼어드는 특수규칙이 있는데 일반규칙만 적용하는 오류. (초안의 `기본값변경`+`특칙누락`을 통합.)
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2026학년도 5번(DMN '유일' 적중정책이 기본값), 2026학년도 6번, 2024학년도 30번
* 판정질문 : "별도 지정이 없으면 기본값은 무엇이며, 이 상황에 일반규칙이 아닌 특칙이 끼어드는가?"
* 예시 : DMN에서 여러 규칙이 동시에 만족될 수 있으면 별도 적중정책이 필요한데 이를 무시하면 기본특칙오인.

# 2. 이항대립

두 입장·이론·개념의 **귀속과 비교**를 판정하는 유형. 사회·규범·철학·사학 지문에 매우 자주 나타나며, 세트 2번(추론·설명형)의 중심이다. (문학·평론의 화자·독자 귀속은 `10. 문학·표현`으로 분리.)

대표 세트: 2024학년도 알베르트/사비니, 2026학년도 스볼릭 민주주의 퇴행 모델, 2025학년도 크리톤 해석 논쟁, 2023학년도 벤야민 폭력론. 출제빈도 **매우높음**.

## #관점귀속

주장·비판·옹호를 누구의 관점에 귀속해야 하는지 묻는 유형.

### 입장전도
A의 주장을 B의 주장으로, B의 주장을 A의 주장으로 바꾸는 오류. (특정 논자가 실제로 취하지 않은 입장을 귀속하는 것은 `3.#추론강도 > 비약추론`으로 처리 — 두 입장의 맞바꿈인지, 근거 없는 귀속인지로 구별.)
* 중요도 : 최상
* 출제빈도 : 매우높음
* 대표기출 : 2024학년도 1번·2번(알베르트/사비니), 2026학년도 7번(스볼릭 모델의 유권자 vs 집권자 트레이드오프 위치), 2025학년도 11번(다수주의 vs 반다수주의)
* 판정질문 : "이 말은 누구의 입장인가?"
* 예시 : 스볼릭 모델에서 '민주주의 가치/득표 증감'의 귀속 주체를 바꾸면 오답.

### 비판대상오인
어떤 입장이 비판하는 대상과 옹호하는 대상을 잘못 바꾸는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2015학년도 3번(유종원의 비판 대상은 예·형 자체가 아니라 진자앙의 조치), 2022학년도 2번, 2019학년도 29번
* 판정질문 : "이 비판은 무엇을 겨냥하는가?"
* 예시 : 유종원은 예와 형을 부정하는 것이 아니라 정문과 사형을 동시에 적용하려는 조치를 비판한다.

## #비교대조

두 개념·이론의 공통점·차이점·비교축을 정확히 잡는 유형.

### 공통점과잉
일부 공통점을 전체 동일성으로 확대하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2025학년도 8번(아테네 소년애 = 스파르타 소년애의 기능으로 동일시), 2015학년도 18번, 2017학년도 27번(비개념주의·개념주의의 공통 동의점)
* 판정질문 : "같은 점은 어디까지인가?"
* 예시 : 두 입장이 같은 문제를 다룬다고 해서 원인·해법·평가 기준까지 같은 것은 아니다.

### 차이점삭제
두 입장의 핵심 차이를 지워 동일한 입장처럼 서술하는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2025학년도 3번(불워-리턴의 공감 유발 vs 새커리의 공감 차단 — 정반대 효과), 2021학년도 1번, 2022학년도 5번(㉠과 ㉡의 자연 목적론 차이)
* 판정질문 : "이 지점이 바로 차이점인가?"
* 예시 : 범죄를 사회경제적 산물로 본 입장과 개인의 병증으로 본 입장은 정반대다.

### 비교축오류
비교해야 할 기준(축)을 잘못 잡아 정오가 갈리는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2017학년도 1번(윤리적 평가축 vs 법적 평가축), 2024학년도 13번, 2026학년도 30번(㉠·㉡ 제정/공화제)
* 판정질문 : "두 대상을 어떤 기준으로 비교해야 하는가?"
* 예시 : '가치'라는 단어를 공유해도 경제학적 축(시장실패·편익)과 사회학적 축(규범 가치)은 다르다.

# 3. 필연성과 범위

선택지가 지문에서 허용된 **추론 강도**나 **양·정도의 범위**를 넘거나 낮추는 유형. "반드시·모두·항상·일 수 있다·추론할 수 있다" 같은 표현이 신호이다. 확인형·추론형 전반에 광범위하게 걸린다.

출제빈도 **매우높음**(특히 `미언급·반대확정`과 `비약추론`은 내용일치·추론 문항의 핵심 함정).

## #추론강도

가능성·필연성·확정 가능성의 강도를 판정하는 유형.

### 양상강도오판
지문상 가능한(개연적) 내용을 반드시 그렇다고 확정하거나, 반대로 반드시 성립하는 결론을 단순 가능성으로 낮추는 오류. (초안의 `가능의필연화`+`필연의가능화` 통합.)
* 중요도 : 상
* 출제빈도 : 높음
* 대표기출 : 2024학년도 23번, 2025학년도 5번, 2026학년도 8번
* 판정질문 : "지문은 가능성을 말했는가, 필연을 말했는가?"
* 예시 : 연구 과정에서 착안된 사실을 특정 결과로 확정하면 가능의 필연화, 조건 충족 시 반드시 나오는 결론을 '그럴 수도 있다'로 낮추면 필연의 가능화.

### 미언급·반대확정
제시문에 근거가 전혀 없는 내용을 사실로 단정하거나, 제시문과 정반대되는 내용을 확정하는 오류. **내용일치(확인형) 문항의 표준 함정.**
* 중요도 : 최상
* 출제빈도 : 매우높음
* 대표기출 : 2025학년도 1번, 2026학년도 16번(기울기≠0이면 인과관계가 있다는 서술을 지문이 부정), 2023학년도 7번
* 판정질문 : "이 문장이 제시문에 실제로 있는가, 오히려 반대로 말하고 있지 않은가?"
* 예시 : 지문이 "역반응은 비자발적"이라 했는데 "자발적으로 일어난다"고 하면 반대확정.

### 비약추론
제시문 문장들로부터 논리적으로 따라오지 않는 결론을 확정하는 오류(추론형의 표준 함정). 특정 논자가 취하지 않은 입장을 귀속하는 경우도 여기에 포함.
* 중요도 : 최상
* 출제빈도 : 매우높음
* 대표기출 : 2024학년도 8번, 2025학년도 17번, 2026학년도 26번
* 판정질문 : "그 결론이 제시문 문장들만으로 논리적으로 따라오는가?"
* 예시 : 사비니의 입장이 직접 제시되지 않은 법형성 상황에서 '규범주의 포기'를 도출하면 비약추론.

## #양·정도

전부/일부, 정도, 최소/최대의 범위를 잘못 잡는 유형.

### 전부일부혼동
일부 사례·속성에 대한 진술을 전체에 일반화하거나 전체 진술을 일부로 축소하는 오류(양화 오류: 전칭↔특칭). (한 대상의 부분↔전체 구성은 `4.#구성관계`, 집합↔원소는 `4.#층위관계`로 구별.)
* 중요도 : 상
* 출제빈도 : 높음
* 대표기출 : 2024학년도 4번(레코드가 식별자·준식별자'만'으로 구성된다고 축소), 2025학년도 27번("모든 유형의 모순된 읽기"로 과잉 일반화), 2021학년도 1번
* 판정질문 : "이 진술은 전체에 관한 것인가, 일부에 관한 것인가?"
* 예시 : "모든 X는 …"과 "어떤 X는 …"의 혼동.

### 정도·극값과장
높음/낮음·강함/약함의 정도를 지문보다 과하게 확정하거나, 최솟값·최댓값·평균·총량·비율의 기준을 혼동하는 오류. (초안의 `정도범위과장`+`최소최대혼동` 통합.)
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2024학년도 5번·6번(동질집합의 최솟값·k값과 잔존율), 2026학년도 6번, 2024학년도 8번
* 판정질문 : "제시문이 말한 정도/기준값보다 강하거나 다른 값으로 말하고 있지 않은가?"
* 예시 : k값 변화와 동질집합 크기의 '최솟값', 레코드 '잔존율'을 구분해야 한다.

# 4. 부분과 전체

**제시문 내부의 개념 층위**에서 구성(부분↔전체)과 층위(집합↔원소, 중간산물↔최종산물)를 혼동하는 유형. (같은 방향 논리라도 주어진 도식을 읽는 것이면 `8`, 새 사례에 적용하는 것이면 `9`로 보낸다 — §0.4.)

대표 세트: 2019학년도 온톨로지(상위/하위 개념·속성 상속), 2024학년도 비식별화(레코드의 구성), 2021학년도 프로세스 마이닝(로그→모델). 출제빈도 **높음**.

## #구성관계

한 대상의 부분과 전체 사이 속성 귀속을 판정.

### 부분의전체화
일부 구성요소의 속성을 전체의 속성으로 일반화하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2024학년도 4번, 2026학년도 5번(하위 노드가 상위 결정에 영향을 준다고 해서 모든 입력이 최상위 노드에 직접 연결되는 것은 아님), 2019학년도 24번
* 판정질문 : "이 속성은 부분의 것인가, 전체의 것인가?"

### 전체의부분화
전체 체계의 속성을 특정 부분에도 그대로 부여하는 오류.
* 중요도 : 중
* 출제빈도 : 중간
* 대표기출 : 2017학년도 3번, 2024학년도 4번(데이터 집합의 속성과 개별 레코드의 속성 혼동), 2021학년도 2번
* 판정질문 : "전체에 관한 설명을 이 부분에도 적용할 수 있는가?"

## #층위관계

집합/원소, 데이터/모델/사례, 중간산물/최종산물 등 분석 층위를 판정.

### 집합원소혼동
집합 전체와 그 원소, 후보군과 선택 결과, 속성값과 속성 집합을 혼동하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2019학년도 22번·24번(개념과 개체, ⊤/⊥가 상속하는 속성 집합), 2024학년도 6번(동질집합의 수 vs 레코드 수), 2021학년도 3번
* 판정질문 : "선택지가 말하는 단위는 집합인가 원소인가?"

### 단계산물혼동
절차의 중간 산물과 최종 산물, 입력과 출력, 모델과 로그를 혼동하는 오류(**제시문 개념 층위에 한정**; 도식 판독은 `8.입출력방향오류`, 새 사례 대응은 `9.모델단계오류`).
* 중요도 : 상
* 출제빈도 : 높음
* 대표기출 : 2021학년도 1번·2번(프로세스 발견은 로그에서 모델을 도출), 2023학년도 8번(NLS/NES의 이동 방향), 2020학년도 8번(단백질 합성·이동 단계)
* 판정질문 : "이 산물은 어느 단계에서 나온 것인가?"

# 5. 인과와 변수관계

원인·결과의 방향, 매개 고리, 상관과 인과의 구분, 그리고 **지문이 서술한** 변수 간 증감·단조 관계를 판정하는 유형. (주어진 그래프에서 값·기울기를 읽는 것은 `8.그래프판독`으로 구별.)

대표 세트: 2026학년도 제도와 성장(도구변수·역인과), 2023학년도 세대 보수화(생애주기 vs 코호트), 2015학년도 차선/관세동맹. 출제빈도 **높음**.

## #인과판정

원인·결과의 방향, 매개, 상관/인과를 구분.

### 인과방향전도
원인과 결과를 뒤집는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2026학년도 16번·17번(좋은 제도→성장 vs 성장→제도 개선의 역인과), 2024학년도 13번, 2023학년도 13번
* 판정질문 : "A 때문에 B인가, B 때문에 A인가?"

### 중간고리누락
원인에서 결과로 가는 매개 단계를 생략하여 결론을 성립시키는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2015학년도 5번(관세동맹의 효율성 판단은 무역전환·무역창출 효과의 크기 비교라는 매개를 거쳐야 함), 2024학년도 5번, 2026학년도 8번
* 판정질문 : "A에서 C로 가려면 B라는 매개가 필요한가?"

### 상관의인과화
함께 나타나는 현상이나 수치 관계를 인과관계로 단정하는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2026학년도 16번·18번(도구변수·기울기≠0을 곧 인과로 단정하는 함정), 2023학년도 13번·15번(생애주기 효과와 코호트 효과의 분리), 2024학년도 23번
* 판정질문 : "이 자료가 인과를 보여 주는가, 단순한 동반 변화를 보여 주는가?"

## #증감판정

증가·감소·유지·단조성·비례/반비례 관계를 판정(지문 서술 기준).

### 증감반전
증가해야 할 것을 감소한다고, 감소해야 할 것을 증가한다고 하는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2024학년도 5번(k값을 낮추면 잔존율은 증가·유지), 2025학년도 18번(솔로우 모델에서 저축률 하락 후 소비의 변화 방향), 2020학년도 25번·27번(궤도-속력 관계)
* 판정질문 : "변수가 커지면 결과는 증가·감소·유지 중 무엇인가?"

### 단조성오판
어떤 변수가 한 방향으로만 변한다고 잘못 판단하는 오류. (파레토·최적화 구조 특유의 '조건 수↑=효율↑' 착각은 `6.#차선판정`으로 구별.)
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2015학년도 4번, 2022학년도 17번(K-민즈가 항상 전체 최적해를 보장하지는 않음), 2024학년도 6번
* 판정질문 : "값이 커지면 항상 좋아지는가/단조적으로 변하는가?"

# 6. 문제해결도식

복수 가치·복수 조건이 충돌하는 상황에서 '좋은 선택·차선·비용편익·최적화'를 판정하는 유형. 규범·경제 지문에 자주 등장한다. (본 대분류는 규모가 작고 `5.#증감판정`과 경계가 있으므로 §검토 및 판단 참고.)

대표 세트: 2015학년도 차선의 문제, 2026학년도 스볼릭 모델(트레이드오프), 2023학년도 제도가능곡선(무질서 비용 vs 독재 비용). 출제빈도 **중간**.

## #상충구조

서로 다른 가치·효용·비용의 충돌을 선택지가 삭제하거나 한쪽만 반영하는 유형.

### 트레이드오프삭제
두 가치·효용의 상충을 지우고 한쪽만 보는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2026학년도 7번(유권자·집권자가 직면한 상충), 2023학년도 19번·21번(자유방임의 무질서 비용과 국가 개입의 독재 비용), 2018학년도 34번
* 판정질문 : "지문은 두 가치 사이의 충돌을 말하고 있지 않은가?"

### 비용편익편측
비용과 편익을 함께 비교해야 하는데 한쪽 요소만 보는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2015학년도 5번(관세동맹은 무역창출 효과만으로 판단할 수 없고 무역전환 효과와 비교해야 함), 2024학년도 8번, 2024학년도 13번
* 판정질문 : "편익만 보았는가, 비용도 보았는가?"

## #차선판정

부분 최적과 전체 최적의 관계, 충족 조건 수와 효율의 관계를 판정(파레토·차선의 문제 특유 구조).

### 부분·전체최적혼동
한 부문의 개선이 곧 전체 개선을 보장한다고 보는 오류.
* 중요도 : 상
* 출제빈도 : 낮음(세트는 적으나 개념 중요도 높음)
* 대표기출 : 2015학년도 4번·6번(하나의 왜곡 시정이 새 왜곡을 낳으면 전체 효율은 반드시 개선되지 않음), 2022학년도 17번(국지해 vs 전체 최적해)
* 판정질문 : "부분의 개선이 전체 효율성 개선으로 자동 연결되는가?"

### 조건수효율착각
충족되는 조건의 수가 많을수록 무조건 더 낫다고 보는 오류.
* 중요도 : 상
* 출제빈도 : 낮음(2015 차선 세트 집중, 그러나 개념 중요)
* 대표기출 : 2015학년도 4번, 2015학년도 6번
* 판정질문 : "조건 수가 많다는 것만으로 더 좋은가?"
* 예시 : 파레토 최적 조건이 일부 깨진 상태에서는 조건 수의 단순 비교로 효율성을 판단할 수 없다.

# 7. 강화약화

어떤 자료·사례·논거가 주장을 **강화하는지 약화하는지**, 진짜 반례인지, 상대 주장을 정확히 겨냥하는지 판정하는 유형. (자료·사례를 본문 틀에 '정확히 대입·분류'하는 것은 `9. 사례적용`으로 구별 — §0.4. 초안의 `사례역전`·`보충근거오판`을 여기로 통합.)

대표 세트: 2015학년도 관세동맹(바이너 지지 사례), 2026학년도 사토리오(도덕 책임 반론), 2024학년도 흄 논쟁 평가, 2019학년도 각종 비판 문항. 출제빈도 **높음**.

## #강화약화판정

자료·사례가 주장을 강화하는가 약화하는가를 판정.

### 강화약화전도
강화 근거를 약화 근거로(또는 그 반대로) 판정하는 오류. 지지 사례를 반례로 보는 경우 포함.
* 중요도 : 상
* 출제빈도 : 높음
* 대표기출 : 2015학년도 5번(무역전환 효과가 큰 사례라야 바이너 입장을 지지), 2024학년도 26번, 2019학년도 20번
* 판정질문 : "이 정보는 주장의 어느 고리를 보강하는가, 흔드는가?"

### 무관근거오판
주장과 논리적으로 무관한 자료를 강화·약화 근거로 오인하는 오류(논점 무관).
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2019학년도 29번(㉠ 비판의 논거로 부적절한 것), 2024학년도 18번, 2023학년도 12번
* 판정질문 : "이 근거가 정말 그 주장의 참·거짓에 영향을 주는가?"

## #반박구조

반례·반박이 상대 주장을 정확히 겨냥하는지 판정.

### 실질반례오인
반례처럼 보이는 사례가 실제로는 주장을 반박하지 못하거나 오히려 지지하는 경우.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2015학년도 5번, 2026학년도 24번(사토리오에 대한 반론이 실제로 성립하는지), 2023학년도 30번
* 판정질문 : "이 사례가 정말 결론을 무너뜨리는가?"

### 논증구조반박
상대 논증의 **구조적 약점**(숨은 전제 누락, 선결문제 요구=순환논증, 허수아비 왜곡 등)을 지적하거나 오인하는 오류. (초안의 `전제누락`·`허수아비공격`을 통합.)
* 중요도 : 중
* 출제빈도 : 중간
* 대표기출 : 2026학년도 24번(사토리오가 "증명할 것을 가정한다"는 선결문제 지적), 2022학년도 27번, 2018학년도 23번
* 판정질문 : "상대 논증이 실제로 그 전제를 필요로 하는가/그 말을 했는가?"

# 8. 모형·자료·절차

**주어진 형식적 자료**(표·그래프·수식·알고리즘·의사결정표)를 제시문 원리에 따라 읽는 유형. 최근 과학기술·사회과학·의사결정 세트에서 비중이 커졌다. (자료 없이 지문 개념 층위만 다루면 `4`, 새 사례 대입이면 `9`로 구별.)

대표 세트: 2026학년도 DMN(의사결정표·조건식)·혼합물 부피(그래프), 2024학년도 광역학 치료(생존율 표), 2025학년도 솔로우 모델(그림). 출제빈도 **높음**.

## #자료해석

표·수치·그래프·경계값을 정확히 읽는 유형.

### 표·수치오독
표의 행과 열, 조건과 결과, 실험군과 대조군의 의미를 잘못 읽는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2024학년도 23번(생존율 표에서 빛·감광제·항산화제 유무 혼동), 2013학년도 27번, 2017학년도 20번(변광성 밝기)
* 판정질문 : "이 표에서 행과 열은 각각 무엇을 나타내는가?"

### 그래프판독
주어진 그래프에서 곡선·기울기·교점·구간의 의미를 잘못 읽는 오류(지문 서술이 아니라 **그림 자체**를 읽는 경우).
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2026학년도 27번(분몰 부피 = 접선의 기울기), 2025학년도 17번(솔로우 <그림>의 교점=정태상태 자본량), 2020학년도 29번(나이테 패턴)
* 판정질문 : "이 그래프에서 축·기울기·교점은 무엇을 의미하는가?"

### 경계값오류
이상/초과, 이하/미만, 포함/배제 조건을 잘못 적용하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2026학년도 4번·6번(`>120`·`<=120`·`[a..b]` 구간과 비교식의 구분), 2024학년도 6번
* 판정질문 : "경계값은 포함되는가, 제외되는가? 숫자 구간인가 단순 비교식인가?"

## #절차흐름

알고리즘·모델·의사결정 절차에서 조건 결합과 데이터 흐름을 판정.

### 조건결합오판
주어진 절차·표에서 여러 조건이 모두 충족되어야 하는지(AND), 하나만이면 되는지(OR)를 혼동하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2026학년도 5번·6번(DMN 의사결정표의 조건 열은 AND로 계산), 2018학년도 7번(DNA 컴퓨팅), 2019학년도 8번
* 판정질문 : "조건들이 AND인가 OR인가?"

### 입출력방향오류
**주어진 도식**에서 입력/출력, 하위/상위 노드, 결과 전달의 방향을 뒤집는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2026학년도 5번(하위 의사결정 노드의 결과가 상위 노드의 입력으로 전달), 2019학년도 24번(상속 방향), 2021학년도 1번
* 판정질문 : "정보는 이 도식에서 어느 방향으로 이동하는가?"

# 9. 사례적용

본문의 원리·모델을 <보기>나 선택지의 **새로운 상황에 정확히 대입·분류**하는지 판정하는 고난도 유형. 세트 3번(적용형)의 중심. (사례가 주장을 강화/약화하는지 '평가'하는 것은 `7`로 구별 — §0.4.)

대표 세트: 2026학년도 지구법학(권리주체 위계 대입), 2024학년도 비식별화(k·ℓ 계산), 2020학년도 상복 예송(처/첩 규정 대입), 2025학년도 배아 법(독일법 vs 한국법). 출제빈도 **매우높음**.

## #대입정확성

지문 기준을 <보기> 대상에 적용할 때 대입 대상과 조건 조합이 맞는지 본다.

### 대입대상오류
본문 기준을 <보기>의 엉뚱한 대상에 적용하는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2026학년도 3번(레건·테일러·베리·컬리넌의 권리주체 인정 범위를 판결 [A]~[C]에 대입), 2025학년도 21번(독일법/한국법을 각 사안에 대입), 2020학년도 5번·6번(처/첩 신분에 따른 상복 규정 대입)
* 판정질문 : "본문 기준은 <보기>의 어느 대상에 적용되어야 하는가?"

### 조건조합오류
<보기>의 여러 조건을 일부만 보거나 잘못 결합하여 결론을 내는 오류.
* 중요도 : 최상
* 출제빈도 : 높음
* 대표기출 : 2024학년도 6번(k값·ℓ값·동질집합·잔존율을 동시에 계산), 2026학년도 6번(DMN 단계별 조건 종합), 2024학년도 24번(신물질 X·Y·Z 실험 조건 조합)
* 판정질문 : "<보기> 조건을 모두 조합하면 결론이 어떻게 되는가?"

## #모델대응

<보기> 사례가 본문 모델의 어느 위치·유형에 대응하는지 판정.

### 모델단계오류
<보기> 사례가 본문 모델의 입력/중간 단계/결과 중 어디에 놓이는지 잘못 판정하는 오류(**새 사례를 모델 단계에 대응**; 지문 개념 층위는 `4`, 도식 판독은 `8`).
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2021학년도 3번(대기시간 정보 추가 = '확장' 단계), 2026학년도 6번, 2025학년도 29번(심포니/시네마/조각 모델 대응)
* 판정질문 : "이 사례는 모델의 입력·중간·결과 중 어디에 놓이는가?"

### 유형분류오류
본문이 제시한 여러 유형·범주 중 <보기> 사례가 어디에 속하는지 잘못 분류하는 오류.
* 중요도 : 상
* 출제빈도 : 높음
* 대표기출 : 2015학년도 5번(사례가 무역창출인지 무역전환인지 분류), 2025학년도 12번(X국 상황을 사법심사 네 모델 중 어디로 분류), 2022학년도 15번((가)~(다)를 파시즘 해석 입장으로 분류)
* 판정질문 : "이 사례는 본문의 어느 유형·범주에 속하는가?"

# 10. 문학·표현

대표 세트: 2025학년도 뉴게이트 소설, 2026학년도 노파와 성경([A]~[C] 서사구조), 2022학년도 서사 시점 이론, 2025학년도 희곡 무대화 모델. 출제빈도 **중간**.

## #표현지시

특정 어구나 상징이 문맥에서 무엇을 가리키고 어떤 기능을 하는지 판정.

### 어구지시오인
지문 속 표현이 가리키는 대상을 잘못 파악하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2025학년도 2번('자연의 제일법칙에 입각한 선택지'는 형법 적용 대상 선택지이지 계급적 순응이 아님), 2020학년도 11번(㉠~㉤), 2023학년도 2번
* 판정질문 : "이 표현은 앞뒤 문맥에서 무엇을 지시하는가?"

### 상징·비유해석
상징·비유가 문맥에서 수행하는 기능을 단순 소재나 표면 의미로 축소·오해하는 오류.
* 중요도 : 중
* 출제빈도 : 낮음
* 대표기출 : 2025학년도 2번('봉쇄된 정의'는 참회·감동의 유예를 차단한 방식의 정의), 2024학년도 20번(㉠ 사례 인용 의도), 2023학년도 문학 세트
* 판정질문 : "이 상징은 단순한 대상인가, 논지 전개상의 기능을 하는가?"

## #서사수용

화자·서술시점, 그리고 독자·대중·인물의 수용 효과를 판정.

### 화자시점혼동
화자·서술자·작가·비평가의 서술 위치와 시점을 혼동하는 오류.
* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2022학년도 7번·8번·9번(시점 분류 기준, 일인칭 서술의 정보 관계), 2026학년도 19번·20번([A]~[C]의 서술자·관계), 2013학년도 17번(맥베스 부인)
* 판정질문 : "이 말은 작품 속 인물의 말인가, 서술자·비평가의 말인가?"

### 수용효과오판

* 중요도 : 상
* 출제빈도 : 중간
* 대표기출 : 2025학년도 1번·3번(범죄자에 대한 공감 유발/차단, 수용층 확대에 따른 우려), 2019학년도 12번(<보기> 감상), 2020학년도 12번
* 판정질문 : "이 표현·장치가 독자에게 어떤 효과를 냈는가? 그 반응은 누구의 것인가?"

## 12. 최종 판정 절차

AI는 새 문항을 분류할 때 다음 순서를 따른다.

1. **선택지 단위로 분해한다.** 5지선다는 5개, ㄱ·ㄴ·ㄷ형은 보기 명제 각각을 분류한다.
2. **지문 근거 문장을 찾는다.** 선택지의 주어·술어·조건·범위·평가어에 대응하는 지문 문장을 표시한다.
3. **오답이면 변형 방식을 지목한다.** 확대·축소·반전·누락·치환·과잉확정 중 무엇인지 판단한다.
4. **동점 처리 원칙(§0.4)으로 대분류를 결정한다.** "무엇을 대상으로 추론하는가"(지문 개념 / 주어진 자료 / 새 사례; 이론 입장 / 문학 목소리; 강화·약화 평가 / 대입·분류)를 먼저 확정한 뒤 중간층위·마지막층위를 붙인다.
5. **태그를 1~3개 부착한다.** 가장 직접적으로 정오를 가르는 조작이 주 태그, 적용·자료·귀속 방식이 보조 태그.
6. **대분류 자동연상 금지.** 과학기술 지문이라고 자동으로 `모형·자료·절차`, 문학 지문이라고 자동으로 `문학·표현`, 법·규범 지문이라고 자동으로 `논리구조`가 아니다. 언제나 선택지의 판정 단위가 우선한다.

### 경계 사례 빠른 참조
- "이 사례가 주장을 지지/약화?" → **7. 강화약화**. "이 사례가 본문 어느 유형/단계?" → **9. 사례적용**.
- "지문이 말한 A↑→B의 방향" → **5. 증감판정**. "주어진 그래프의 기울기" → **8. 그래프판독**.
- "입력→출력 방향"이 지문 개념이면 **4**, 주어진 도식이면 **8**, 새 사례면 **9**.
- "누구의 주장인가"가 이론·논증이면 **2**, 화자·독자면 **10**.
- "필요조건/충분조건 뒤집기"는 **1. 조건역전**, "원인/결과 뒤집기"는 **5. 인과방향전도**.
