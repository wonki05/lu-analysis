# 2025학년도 LEET 언어이해 문제지 Markdown 변환 작업 보고서

## 1. 생성한 파일 목록

### Markdown 파일
- `2025_LU_problems.md`

### PNG 이미지 파일
- `2025_LU_images_12_1.png`
- `2025_LU_images_text_16-18_1.png`
- `2025_LU_images_27_1.png`

## 2. 이미지로 캡처한 항목 목록

| 파일명 | 위치 | 이미지로 처리한 이유 |
|---|---|---|
| `2025_LU_images_12_1.png` | 12번 문항 `<보기>` | 두 개의 누적 막대그래프, 범례성 말풍선, 시점별 비율 배치가 문제 풀이에 필요한 시각 정보이므로 PNG로 캡처함. |
| `2025_LU_images_text_16-18_1.png` | 16~18번 공통 제시문 | 생산량·감가상각량·저축량 곡선의 상대 위치와 교점이 문제 풀이에 필요한 그래프이므로 PNG로 캡처함. |
| `2025_LU_images_27_1.png` | 27번 문항 `<보기>` | 트랜잭션 A/B/C의 실행 순서, 데이터베이스 상태 변화, 화살표 방향과 시간 흐름이 핵심인 도식이므로 PNG로 캡처함. |

## 3. 표 처리 목록

| 위치 | 처리 방식 | 이유 |
|---|---|---|
| 6번 문항 `<보기>`의 (나) 표 | Markdown 표 | 효소·기질·산물·포르피린증 종류를 텍스트로 정확히 재현할 수 있는 구조였고, GitHub에서 검색 가능하도록 표를 텍스트화하는 것이 적절하다고 판단함. 일부 셀 내부 줄바꿈은 `<br>`로 반영함. |

## 4. 판단이 필요했던 부분

| 위치 | 선택한 처리 방식 | 이유 | 가능한 대안 |
|---|---|---|---|
| 전체 본문 도서명 표기 | PDF 렌더링상 도서명 괄호를 `『 』`로 전사함. | PDF 텍스트 추출 결과에서는 한컴 계열 특수 글리프가 ``처럼 보였으나, 렌더링 이미지 기준으로 책명 표지로 판독됨. | 원문 글리프를 그대로 보존하는 방식도 가능하나 GitHub/AI 검색성은 `『 』`가 더 높음. |
| 10~12번, 28~30번의 작은 박스형 용어 | `<kbd>네 가지 모델</kbd>`, `<kbd>심포니 모델</kbd>` 등으로 처리함. | 문항에서 직접 참조되는 작은 박스 표시이므로 텍스트를 유지하면서 박스형 표시를 보존할 필요가 있음. | 일반 텍스트로만 표기할 수 있으나 원문의 참조 표지가 약해짐. |
| 16~18번 제시문의 수식과 변수 | `생산량($y$)`, `자본량($k$)`, `$$\Delta k = i - dk$$`처럼 LaTeX로 처리함. | 원문 수식은 텍스트로 정확히 전사 가능한 단순 수식이며, 지침상 LaTeX 수식 표현이 적절함. | 수식을 이미지로 캡처할 수 있으나 검색성과 재사용성이 떨어짐. |
| 16~18번 제시문의 `[A]` 표지 | `[A]`를 해당 단락 첫머리에 배치함. | 원문에서는 문단 좌측/하단의 표지로 보이지만 Markdown에서는 해당 단락을 참조하는 표지임을 가장 명확히 드러내는 위치가 단락 첫머리라고 판단함. | 단락 앞 별도 줄에 `[A]`만 배치하거나, 인용문 블록으로 표시할 수 있음. |
| 부정 발문 밑줄 | `적절하지 <u>않은</u>`, `일치하지 <u>않는</u>`처럼 부정어 밑줄을 HTML underline으로 반영함. | PDF 렌더링 이미지에서 부정어에 밑줄 강조가 확인되어 원문 강조를 보존함. | 밑줄을 제거할 수 있으나 원문의 강조 정보가 사라짐. |

## 5. 판독이 불확실했던 부분

- 최종 본문에 남겨 둔 판독 불확실 표지는 없음.
- 다만 PDF 내 한컴 특수 글리프가 텍스트 추출상 `` 등으로 표시되는 부분은 렌더링 이미지를 기준으로 도서명 괄호 `『 』`로 통일하였다.

## 6. 최종 검수 결과

| 항목 | 결과 |
|---|---|
| PDF 전체 페이지 수 | 15쪽 확인 |
| Markdown 파일명 | `2025_LU_problems.md` 형식 확인 |
| 전체 문항 수 | 1번부터 30번까지 30문항 확인 |
| 문항 번호 형식 | `## 01`부터 `## 30`까지 두 자리 형식 확인 |
| 각 문항 선택지 수 | 모든 문항 (1)~(5) 5개 선택지 확인 |
| 공통 제시문 범위 | [01-03], [04-06], [07-09], [10-12], [13-15], [16-18], [19-21], [22-24], [25-27], [28-30] 확인 |
| 표 반영 여부 | 6번 `<보기>` 표를 Markdown 표로 반영 |
| 수식 반영 여부 | 16~18번 제시문의 변수 및 식을 LaTeX로 반영 |
| 이미지 링크 정상 여부 | Markdown 내 3개 이미지 링크가 실제 PNG 파일명과 일치함 |
| 이미지 링크 형식 | 모두 `./파일명.png` 형식 확인 |
| 제외 대상 제거 여부 | 표지성 제목, 교시/머리말, 성명·수험번호란, 홀수형 표시, 페이지 번호, 답안 작성 안내, 마지막 확인 사항 제거 |
| 원문 외 해설·정답·주석 포함 여부 | Markdown 본문에는 원문 문제 구조화 외 내용 없음 |
