## 목적
`### 선택지` 블록의 선택지들이 빈 줄 없이 연속돼 있어 Markdown preview에서 한 문단으로 붙어 렌더링되는 문제를 해결. 선택지 사이에 빈 줄을 1개씩 삽입해 각 선택지가 별도 문단으로 보이도록 함.

## 방법 / 안전장치
- `### 선택지` 블록 **내부**의, 줄 시작이 `(숫자)`인 선택지 라인 **앞에만** 빈 줄을 삽입.
- 선택지 외 텍스트(제시문, stem, `### 보기`, `<u>`/`<kbd>`, 표 등)는 **미변경**. 텍스트 라인은 삭제·수정 없이 빈 줄만 추가.
- 표 형식 선택지(`| (1) | … |`, 줄 시작이 `|`)는 패턴에 걸리지 않아 **제외**(표 무손상). → 2013년 1·12·27번(어휘 표 문항)은 미수정.
- 적용 후 검증: `<u>`/`<kbd>`/`**` 태그 카운트·이미지 링크·문항 수가 적용 전과 **동일**(텍스트 무변경 확인).
