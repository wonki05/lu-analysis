## 6. 판단이 필요했던 부분

| 위치 | 선택한 처리 방식 | 이유 | 가능한 대안 |
|---|---|---|---|
| 13~15번 공통 제시문 그림 | 공통 제시문 이미지 파일 `2024_LU_images_text_13-15_1.png`로 유지 | 그림은 13~15번 문항 공통 제시문 안에서 사회성과 측정 방식을 설명하는 시각 자료이므로 개별 문항 이미지가 아니라 공통 제시문 이미지로 보는 것이 적절하다. | 개별 문항 이미지로 분류할 수도 있으나, 원문상 13~15번 공통 제시문 내부 자료이므로 채택하지 않았다. |
| 13~15번 공통 제시문 그림 캡션 | 이미지 crop에 포함 | `<그림> 이해관계자를 고려한 사회성과 측정`은 도식의 제목이자 원문 시각 자료와 결합된 캡션이므로 이미지에 포함했다. | 캡션을 Markdown 텍스트로 분리 전사할 수 있으나, 원문에서는 도식 바로 아래에 결합되어 있어 이미지에 포함하는 편이 자연스럽다. |
| 밑줄 표시된 참조 표현 | Markdown 본문에 `<u>...</u>`로 반영 | 원문에서 문항이 참조하는 ㉠, ㉡, ㉢, ㉣, ㉤, ⓐ, ⓑ, ⓒ 등의 표시가 밑줄과 결합된 부분은 문제 풀이상 참조 범위가 될 수 있어 보존했다. | 밑줄 없이 기호만 둘 수 있으나, 원문의 강조 표시 보존 원칙에 따라 채택하지 않았다. |

## 7. 판독이 불확실했던 부분

- 최종 본문에 반영하면서 판독이 불확실하다고 남긴 글자는 없음.
- 원본 PDF 렌더링 이미지 기준으로 기존 Markdown과 대조했고, 표의 숫자·기호·선택지 번호·부정 표현을 재검토했다.
